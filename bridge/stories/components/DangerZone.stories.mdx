# Danger Zone

The danger zone is a component that emphasizes on the terminal operation of deleting a project or service.

## Project

Let's define the danger zone for a project:

<Canvas>
  <Story name="Project">
    {{
      template: `
        <ktb-danger-zone [data]="${"{ name: 'project-1', type: 'project' }"}"></ktb-danger-zone>
      `
    }}
  </Story>
</Canvas>
